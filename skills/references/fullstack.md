# Full Stack Developer — MNC Skill Reference

## Frontend
| Skill | SDE-1 | SDE-2 |
|---|---|---|
| React (hooks, context, performance) | Proficient | Expert — optimization, custom hooks, patterns |
| TypeScript | Working knowledge | Strong typing, generics, utility types |
| State management (Redux / Zustand / Jotai) | One library well | Multiple, knows when to use which |
| CSS / Tailwind / CSS-in-JS | Proficient | Responsive, accessible, design systems |
| Next.js (SSR, SSG, ISR) | Aware | Proficient — knows trade-offs |
| Testing (Jest, RTL, Cypress) | Unit tests | Integration + E2E |
| Web performance (Core Web Vitals) | Aware | Proficient — can debug and fix |
| Accessibility (WCAG) | Aware | Implements it |

## Backend
| Skill | SDE-1 | SDE-2 |
|---|---|---|
| Node.js / Express or FastAPI | Proficient | Production-grade patterns |
| REST API design | Proficient | Versioning, pagination, idempotency |
| GraphQL | Awareness | Proficient |
| Auth (JWT, OAuth2, sessions) | Implements correctly | Designs auth systems |
| Database (PostgreSQL + one NoSQL) | CRUD + indexing | Query optimization, schema design |
| Caching (Redis) | Awareness | Cache invalidation strategies |
| Message queues (Kafka / RabbitMQ) | Awareness | Used in production context |
| Docker | Can Dockerize an app | Multi-stage builds, docker-compose |
| CI/CD (GitHub Actions / Jenkins) | Awareness | Sets up pipelines |
| Cloud (AWS / GCP basics) | EC2, S3, RDS | Lambda, CloudFront, managed services |

## Architecture Knowledge
- MVC, clean architecture, layered architecture
- Monolith vs microservices trade-offs
- API gateway patterns
- 12-factor app principles

## Must-Have for Tier 1
- Demonstrated full ownership of a feature from DB schema → API → UI → deployment
- Performance debugging experience (slow queries, render bottlenecks)
- System design: design Instagram feed, design URL shortener

## Green Flags on Resume
- Open source contributions
- Side projects with real users (not just GitHub repos)
- Blog posts or talks on technical topics
- Monorepo experience (Turborepo, Nx)
