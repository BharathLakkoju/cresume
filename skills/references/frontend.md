# DSA & Problem Solving Expectations

---

## SDE-1 Expectations

### Tier-1 (Google, Microsoft, Amazon, Meta)

- **LeetCode target**: 150–200 problems (70% medium, 30% hard)
- **Must-know topics**: Arrays, strings, linked lists, trees, graphs, DP (basic), sliding window, two pointers, binary search, BFS/DFS, hash maps, heaps
- **Time complexity**: Must explain Big-O for every solution
- **Rounds**: 4–5 DSA rounds, 45 min each, expected to code clean + optimal solution
- **Reality check**: Google is the hardest. Amazon focuses on medium-hard graphs + DP. Microsoft tends toward medium difficulty.

### Tier-2 (Flipkart, Swiggy, Razorpay, PayPal)

- **LeetCode target**: 100–150 problems (mostly medium)
- **Must-know topics**: Arrays, strings, trees, graphs, DP (medium), greedy, stack/queue, sliding window
- **Rounds**: 2–3 DSA rounds, cleaner code expected, edge cases matter

### Tier-3 (Atlassian, SAP, Oracle, Cisco)

- **LeetCode target**: 75–100 problems (easy-medium)
- **Must-know topics**: Arrays, strings, trees, hash maps, basic sorting, linked lists
- **Rounds**: 1–2 DSA rounds, correctness over optimization

---

## SDE-2 Expectations

### Tier-1

- **Same DSA breadth as SDE-1 but faster** — expected to solve medium in 15–20 min
- **Hard problems**: Must attempt, partial solutions with explanation accepted at some companies
- **Additional**: Advanced graph algorithms (Dijkstra, Union-Find, Topological sort), Advanced DP (interval, bitmask), Segment trees (for Google)
- **Trade-off discussions**: Space vs time, iterative vs recursive

### Tier-2

- DSA bar similar to Tier-1 SDE-1 but emphasis shifts toward system design
- Clean code, good variable naming, test cases expected

### Tier-3

- DSA bar lighter at SDE-2 — more emphasis on design and experience
- Medium problems expected, hard is a bonus

## Study Plan

### 3-Month Plan (SDE-1, Tier-1 target)

- **Month 1**: Arrays, strings, linked lists, trees, binary search (NeetCode 150 — first 75)
- **Month 2**: Graphs, DP, heaps, intervals (NeetCode 150 — second 75)
- **Month 3**: Hard problems, timed practice, mock interviews
