# AI / ML Engineer — MNC Skill Reference

## Foundations (Non-Negotiable)

| Skill                                          | SDE-1               | SDE-2                            |
| ---------------------------------------------- | ------------------- | -------------------------------- |
| Python (numpy, pandas, scikit-learn)           | Proficient          | Expert                           |
| Linear algebra, probability, statistics        | Solid understanding | Deep — can derive from scratch   |
| ML fundamentals (supervised, unsupervised)     | Proficient          | Deep + can explain trade-offs    |
| Model evaluation (precision/recall, AUC, etc.) | Proficient          | Proficient + knows failure modes |
| Feature engineering                            | Basic               | Advanced — domain-aware          |

## Deep Learning

| Skill                         | SDE-1                       | SDE-2                              |
| ----------------------------- | --------------------------- | ---------------------------------- |
| PyTorch or TensorFlow         | One proficiently            | Both familiarity, one deeply       |
| CNNs, RNNs, Transformers      | Conceptual + implementation | Production-grade                   |
| Fine-tuning pretrained models | Has done it                 | Optimizes for inference            |
| Loss functions, optimizers    | Understands                 | Can design custom ones             |
| Training pipelines            | Can write one               | Scales them (distributed training) |

## LLMs & Generative AI (Now Expected)

| ----------------------------------------------- | ----------------------- | ---------------------- |

| Fine-tuning LLMs (LoRA, QLoRA)                  | Awareness               | Has done it            |

| Agent frameworks (LangGraph, AutoGen)           | Awareness               | Proficient             |

## MLOps & Deployment

| Skill                                        | SDE-1             | SDE-2                          |
| -------------------------------------------- | ----------------- | ------------------------------ |
| Model serving (FastAPI, TorchServe, BentoML) | Basic             | Production-grade               |
| Docker + containerization                    | Dockerizes models | Optimized inference containers |
| Cloud ML (AWS SageMaker / GCP Vertex AI)     | Awareness         | Proficient                     |
| Experiment tracking (MLflow, W&B)            | Uses one          | Sets up for team               |
| CI/CD for ML pipelines                       | Awareness         | Implements                     |
| Monitoring (data drift, model degradation)   | Awareness         | Implements                     |

## Must-Have for Tier 1

- End-to-end ML project: data collection → training → serving → monitoring
- Demonstrated work with LLMs in production (not just notebooks)
- System design: design a recommendation system, design a real-time fraud detector

## Green Flags on Resume

- Kaggle (top 10% or higher in a competition)
- Research paper or blog post on a technique
- Open-source AI tool with GitHub stars
- Inference optimization work (quantization, ONNX export)
- Real product impact numbers ("model improved CTR by 12%")
